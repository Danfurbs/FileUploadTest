Safe upload demo

This demo uses a single, hidden <input type="file"> that is triggered when you click "Select a file". The chosen file is read with FileReader.readAsText() so we can confirm the browser can safely load it without relying on drag-and-drop or the File System Access API.

Reads the selected file as text and shows a short preview. Edge 143 has a crash bug when using FileReader, so this demo auto-switches to a safer path on that version.
