Simple file upload tester

This page triggers a hidden file input, reads the selected file as text with FileReader.readAsText(), and shows a short preview so you can verify browser uploads work.

If browser uploads are blocked by DLP, consider running this page inside a lightweight desktop wrapper (e.g. WebView2, Electron, or Tauri). It keeps the HTML the same but routes file picking through native APIs instead of the browser sandbox.
